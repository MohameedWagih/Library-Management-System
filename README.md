# 📚 ReadHub - Library Management System

A robust, dynamic, and secure Library Management System built with **Django** and **Vanilla JavaScript**. Designed with a focus on clean code, seamless user experience (UX), and secure data handling.

## ✨ Key Features

* **🔍 Live Smart Search:** Instant, dynamic search filtering (by title) without page reloads, using optimized JavaScript event listeners.
* **📊 Financial Dashboard:** Real-time tracking and visual representation of profits from rented and sold books.
* **🗂️ Dynamic Category Filtering:** Filter books by categories and availability status (Available, Rented, Sold) on the fly.
* **🛡️ Secure CRUD Operations:** Fully protected views utilizing CSRF tokens and secure object retrieval (404 handling) for adding, updating, and deleting records.
* **💻 Interactive UI:** Responsive and intuitive design built with Bootstrap, custom CSS, and jQuery.

## 🛠️ Tech Stack

* **Backend:** Python, Django 5
* **Frontend:** HTML5, CSS3, JavaScript, jQuery, Bootstrap, Chart.js
* **Database:** SQLite (Development)
